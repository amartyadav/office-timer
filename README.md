# Office Timer
A simple timer utility to manage check-in and check-out times for office employees. This tool helps track working hours and ensures accurate timekeeping.<br/>
Mostly written for my own use, but feel free to use it if you find it helpful!<br/>

## Currently Under Development
This project is still in the early stages of development. I am actively working on adding the basic features of time-keeping. 
Will be adding more features and improving the user experience in the future. Stay tuned for updates!
